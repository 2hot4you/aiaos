## 1. 产品概述

### 1.1 产品定位

AIAOS 是一款面向公司内部员工的 **AI 漫剧生成平台**。用户可在平台上完成从剧本编写、角色/场景设计、分镜生成、视频合成到成片导出的全流程漫剧创作。

### 1.2 核心价值

| 维度 | 说明 |
|------|------|
| **效率提升** | 将传统漫剧制作周期从数周缩短至数小时 |
| **降低门槛** | 非专业人员也能完成专业级漫剧输出 |
| **AI 驱动** | 生文（GPT）、生图（Gemini）、生视频（Sora）三模型联动 |
| **全流程覆盖** | 剧本 → 分镜 → 资产 → 视频 → 成片，一站式完成 |

### 1.3 产品边界

- ✅ 内部员工使用，不对外开放
- ✅ 全流程 AI 辅助漫剧生成
- ❌ 不提供自助注册
- ❌ 不提供忘记密码自助找回
- ❌ 不涉及分发/发布/社交功能

## 5. 页面流程图

### 5.1 主流程

```
[登录页] 
  → 输入用户名+密码 → 验证通过
  → [项目库首页]
      → 新建项目 / 选择已有项目
      → [剧集管理页]
          → 新建季/集 / 选择已有集
          → [工作台 - 剧本与故事]
              → 配置项目参数 + 编写剧本
              → AI 生成分镜剧本
              → [工作台 - 角色与场景]
                  → 确认/调整角色、场景、道具资产
                  → [工作台 - 导演工作台]
                      → 逐个分镜确认 → 生成视频
                      → [工作台 - 成片与导出]
                          → 预览 → 导出
```

### 5.2 管理员流程

```
[登录页] → 管理员登录
  → [项目库首页]（同普通用户）
  → [管理员后台]（顶部导航入口）
      → 用户管理：增删改查员工账号
      → 模型管理：配置 AI 模型连接信息
      → 系统设置：全局参数
      → 组件参数：各功能默认值
```

### 5.3 剧本创作子流程

```
编写/粘贴剧本
  → 选择创作模式（标准/高级）
  → 配置参数（风格、比例、时长、模型等）
  → 点击「生成分镜剧本」
  → 系统调用生文模型（GPT）
  → 输出：分镜列表 + 运镜 + 提示词
  → 自动识别：角色/场景/道具
  → 用户确认 → 进入「角色与场景」
```

### 5.4 资产生成子流程

```
从剧本识别的资产列表
  → 逐个资产：
      → 选择操作：AI 生成 / 从资产库替换 / 手动上传
      → AI 生成：
          → 系统调用生图模型（Gemini）
          → 输出图片预览
          → 可选：服装变体 / 造型九宫格
      → 上传参考图 → 参考生图
      → 用户确认 → 资产入库
```

### 5.5 视频生成子流程

```
分镜列表
  → 逐个分镜：
      → 关联角色/道具/场景资产
      → 确认动作描述 + 台词
      → 配置生成参数（模型、比例、时长、配音）
      → 调用视频生成模型（Sora）
      → 生成完成 → 预览
      → 不满意 → 调整参数 → 重新生成
      → 满意 → 确认
  → 全部分镜确认 → 进入「成片与导出」
```

## 6. 功能模块详细描述

### 6.1 登录系统

#### 页面：登录页 `/login`

| 元素 | 类型 | 说明 |
|------|------|------|
| Logo + 产品名称 | 展示 | AIAOS 品牌标识 |
| 用户名输入框 | Input | 必填 |
| 密码输入框 | Input (password) | 必填 |
| 登录按钮 | Button (primary) | 提交登录 |
| 错误提示 | Toast / Alert | 「用户名或密码错误」 |

#### 交互细节

- 登录成功 → 跳转项目库首页
- 登录失败 → 显示错误提示，不清空用户名，清空密码
- 连续 5 次失败 → 锁定账号 15 分钟（防暴力破解）
- 登录态通过 JWT Token 维持，有效期 24 小时
- Token 过期 → 自动跳转登录页
- 无注册入口、无忘记密码入口

#### 接口需求

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 用户登录 | POST | `/api/v1/auth/login` | Body: `{username, password}` → Response: `{token, user}` |
| 登出 | POST | `/api/v1/auth/logout` | Header: Authorization |
| 刷新 Token | POST | `/api/v1/auth/refresh` | Header: Authorization |

**视觉风格选择器详细说明：**

- 预设风格以 **卡片网格** 展示，每个卡片包含预览图 + 风格名称
- 预设选项：
  - 日式动漫
  - 2D 动画
  - 3D 动画
  - 赛博朋克
  - 油画风格
  - 真人影视

**创作模式说明文案：**

| 模式 | 标签 | 描述 |
|------|------|------|
| 标准模式 | 小说生成分镜 | 适合粘贴小说/章节/大纲，系统理解故事后自动拆分镜头 |
| 高级模式 | 分镜生成分镜 | 适合已有分镜脚本，按条生成，保留镜头顺序、对白、动作、备注 |

#### 6.4.3 右侧 — 剧本编辑器

占据剩余宽度，核心编辑区域。

**编辑器功能：**

| 功能 | 类型 | 说明 |
|------|------|------|
| 文本编辑区 | Rich Text Editor | 支持富文本，标题/段落/粗体/斜体 |
| AI 改写 | Button + Popover | 选中文本 → 点击「AI 改写」→ 显示改写结果 → 确认/取消 |
| AI 续写 | Button | 在光标位置续写内容 |
| 选段改写 | Button + Selection | 选中段落 → 输入改写指令 → AI 返回改写结果 |
| 撤回改写 | Button | 恢复上一次 AI 操作前的内容（支持多级撤回） |
| 字数统计 | Badge | 实时显示当前字数 |
| 自动保存 | 自动 | 编辑后 3 秒无操作自动保存草稿 |

**AI 生成分镜流程：**

1. 用户完成剧本编辑 + 配置参数
2. 点击「生成分镜剧本」按钮
3. 系统组装请求：`系统提示词 + 视觉风格 + 创作模式指令 + 用户剧本`
4. 调用分镜生成模型（GPT）
5. 返回结构化数据：

```json
{
  "storyboard": [
    {
      "shot_number": 1,
      "scene_description": "城市夜景，霓虹灯闪烁",
      "camera_movement": "缓慢推进",
      "dialogue": "这座城市从不入眠...",
      "action": "主角站在天台边缘，风吹起衣角",
      "duration": "4s",
      "script_prompt": "...",
      "visual_prompt": "cyberpunk city at night, neon lights, ..."
    }
  ],
  "identified_assets": {
    "characters": [
      { "name": "主角", "description": "...", "image_prompt": "..." }
    ],
    "scenes": [
      { "name": "城市天台", "description": "...", "image_prompt": "..." }
    ],
    "props": [
```

## 8. 非功能需求

### 8.1 性能

| 指标 | 要求 |
|------|------|
| 页面首屏加载 | ≤ 2s（内网环境） |
| API 响应时间（非 AI） | ≤ 200ms（P95） |
| AI 生文响应 | ≤ 30s（首 token 流式返回 ≤ 3s） |
| AI 生图响应 | ≤ 60s |
| AI 生视频响应 | ≤ 180s |
| 并发用户数 | ≥ 50 |
| 异步任务队列吞吐 | ≥ 100 tasks/min |

### 8.2 安全

| 要求 | 说明 |
|------|------|
| 认证方式 | JWT（HttpOnly Cookie 或 Authorization Header） |
| 密码存储 | bcrypt 哈希（cost ≥ 12） |
| 暴力破解防护 | 连续 5 次失败锁定 15 分钟 |
| API Key 存储 | AES-256 加密存储 |
| 权限控制 | 中间件层 RBAC 校验 |
| HTTPS | 生产环境强制 HTTPS |
| CORS | 仅允许同域 |
| SQL 注入防护 | ORM 参数化查询 |
| XSS 防护 | CSP + 输入过滤 + 输出转义 |
| CSRF 防护 | Token 机制 |
| 文件上传 | 类型白名单 + 大小限制 + 病毒扫描（可选） |
| 日志审计 | 记录登录、管理操作、AI 调用日志 |

### 8.3 可用性

| 要求 | 说明 |
|------|------|
| 响应式设计 | 最低支持 1280px 宽度 |
| 浏览器兼容 | Chrome ≥ 120、Edge ≥ 120、Safari ≥ 17 |
| 错误处理 | 所有 AI 操作提供明确错误提示和重试机制 |
| 加载状态 | 所有异步操作显示加载状态 / 进度 |
| 自动保存 | 编辑器内容 3 秒无操作自动保存 |
| 离线提示 | 网络断开时提示用户 |

### 8.4 可维护性

| 要求 | 说明 |
|------|------|
| 日志 | 结构化日志（JSON），支持级别控制 |
| 监控 | API 响应时间、错误率、AI 调用统计 |
| 备份 | 数据库每日自动备份 |
| 部署 | Docker 容器化，支持 docker-compose 一键部署 |
| 配置 | 环境变量 + 配置文件，12-Factor |
| API 文档 | Swagger/OpenAPI 自动生成 |

### 8.5 数据

| 要求 | 说明 |

## 10. 术语表

| 术语 | 定义 |
|------|------|
| **分镜（Storyboard Shot）** | 影视创作中的最小镜头单位，包含画面描述、运镜、台词、动作等 |
| **运镜（Camera Movement）** | 镜头的运动方式，如推、拉、摇、移、跟、固定等 |
| **资产（Asset）** | 角色、场景、道具的统称 |
| **资产库（Asset Library）** | 已确认的资产集合，可跨集/跨项目复用 |
| **提示词（Prompt）** | 提供给 AI 模型的指令文本，引导生成结果 |
| **画面提示词（Visual Prompt）** | 用于生图/生视频的描述性文本 |
| **剧本提示词（Script Prompt）** | 用于 AI 理解剧本上下文的提示词 |
| **造型九宫格（Pose Grid）** | 角色 9 个不同角度/表情/姿势的参考图合集 |
| **服装变体（Costume Variant）** | 同一角色的不同服装版本 |
| **风格反推（Style Inference）** | 上传参考图片，由 AI 分析并生成对应风格描述 |
| **SSE（Server-Sent Events）** | 服务端推送技术，用于实时通知任务进度 |
| **级联删除（Cascade Delete）** | 删除父资源时自动删除所有关联子资源 |
| **软删除（Soft Delete）** | 标记为已删除但不实际移除数据，可恢复 |
